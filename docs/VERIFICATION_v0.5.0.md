## 🔍 已知问题

### NL2SQL 小问题

**问题：** WHERE 子句包含多余词汇

**示例：**
```
输入：查找年龄大于 25 的用户
输出：SELECT * FROM users WHERE 查找年龄 > 25
期望：SELECT * FROM users WHERE 年龄 > 25
```

**影响：** 低 - SQL 仍然可以执行（如果列名匹配）  
**计划：** v0.5.1 修复
