## [v0.5.0] - 2026-03-16

### 🎉 新增 - 智能查询助手

#### 🔍 错误诊断
- 新增 `/diagnose` 命令，自动诊断 SQL 执行错误
- 支持 MySQL/PostgreSQL/SQLite 错误代码识别
- 提供详细的解决方案和置信度评分
- SQL 执行出错时自动显示诊断报告

#### ⚡ 查询优化建议
- 新增 `/optimize` 命令，分析 SQL 性能问题
- 检测 10+ 种性能问题（SELECT *、无 WHERE、子查询等）
- 提供优化建议和置信度评分
- 按严重程度分级（Critical/Warning/Info）

#### 🗣️ 自然语言生成 SQL
- 新增 `/nl2sql` 命令，中文自然语言转 SQL
- 支持查询、统计、条件、排序、限制等操作
- 支持中文表名和列名识别
- 输出置信度评分和说明

#### 🌳 查询计划可视化
- 增强 `/explain` 命令输出
- 执行计划树形可视化
- 成本估算和统计信息
- 自动优化建议

#### 🔖 书签管理系统
- 新增 `/bookmark` 命令，保存常用查询
- 支持标签分类管理
- 支持搜索（名称/SQL/描述/标签）
- 支持导出/导入（JSON 格式）
- 使用统计（次数/最后使用时间）
- 内置书签（常用查询模板）

### 📊 技术改进
- 新增 5 个核心模块（~1800 行代码）
- 新增 6 个测试脚本（58 个测试用例）
- 3 次完整验证通过率 100%（90/90 测试）
- 集成到 CLI 命令：`/diagnose`, `/optimize`, `/nl2sql`, `/explain`, `/bookmark`

### 📝 文档
- 新增 `PHASE3_TASKS.md` - 三期任务规划
- 新增 `VERIFICATION_v0.5.0.md` - v0.5.0 验证报告
- 新增 `DEV_v0.5.0*.md` - 开发日志
- 更新 `README.md` - 添加 v0.5.0 功能说明
- 更新 `USAGE.md` - 添加智能功能使用指南
- 新增 `RELEASE_v0.5.0.md` - v0.5.0 发布说明

### 🧪 测试

## 版本说明

### 语义化版本
- **主版本号 (Major)** - 不兼容的 API 更改
- **次版本号 (Minor)** - 向后兼容的功能新增
- **修订号 (Patch)** - 向后兼容的问题修正

### 发布类型
- **Alpha** - 内部测试版本
- **Beta** - 公开测试版本
- **RC** - 候选发布版本
- **Release** - 正式版本
