```
╔══════════════════════════════════════════════════════════╗

## 2. 命令提示界面 (输入 `/` 后)

```
sql> /
可用命令:
  /config      配置管理      /connect   连接数据库
  /list        列出所有表    /desc      查看表结构
  /run         执行 SQL 文件  /history   历史记录
  /export      导出结果      /format    设置输出格式
  /help        帮助信息      /quit      退出

输入 / 后跟命令首字母可快速筛选，如 /c 显示 config/connect
```

**筛选后:**

```
sql> /c
匹配命令:
  /config      配置管理 (add/list/edit/remove/test)
  /connect     连接数据库实例 (别名：/co, /check-out)

sql> /config █
```

```
sql> SELECT
   ->     u.id,
   ->     u.username,
   ->     COUNT(p.id) AS post_count
   -> FROM users u
   -> LEFT JOIN posts p ON u.id = p.user_id
   -> GROUP BY u.id, u.username
   -> ORDER BY post_count DESC
   -> LIMIT 10;
   ┌────┐
[等待更多输入，以分号结束]
   -> ;
