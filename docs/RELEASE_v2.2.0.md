# DBManager v2.2.0 发布说明

**发布日期：** 2026-03-21  
**版本：** v2.2.0  
**主题：** 插件生态系统

---

## 🎉 重大更新

v2.2.0 引入了插件生态系统功能：

1. **插件市场** - 浏览和安装插件
2. **插件管理** - 安装/卸载/启用/禁用
3. **插件更新** - 自动检查更新
4. **插件搜索** - 搜索和分类

---

## 🔌 插件市场

### 核心功能

**插件浏览：**
- 查看所有可用插件
- 按分类浏览
- 搜索插件
- 查看插件详情

**插件信息：**
- 插件名称和版本
- 描述和图标
- 评分和下载量
- 大小和分类
- 标签

---

## 📦 插件管理

### 安装插件

**步骤：**
1. 导航到"插件"页面
2. 浏览或搜索插件
3. 点击"安装"按钮
4. 等待下载完成
5. 插件自动启用

### 管理插件

**操作：**
- 启用/禁用插件
- 卸载插件
- 检查更新
- 查看插件信息

### 工具类

---

## 🔧 技术实现

### 核心库

| 库 | 用途 |
|------|------|
| pluginManager.ts | 插件管理逻辑 |

### 核心函数

**pluginManager.ts:**
- `getPluginMarketplace()` - 获取市场数据
- `getInstalledPlugins()` - 获取已安装插件
- `installPlugin()` - 安装插件
- `uninstallPlugin()` - 卸载插件
- `togglePlugin()` - 切换插件状态
- `checkPluginUpdates()` - 检查更新
- `searchPlugins()` - 搜索插件

## ⚠️ 已知限制

### 当前限制
1. **插件数量** - 内置插件有限
2. **第三方插件** - 暂不支持
3. **插件沙箱** - 简化实现

### 待优化
1. 第三方插件支持

4. 插件评价系统
