**新增 6 个命令：**
- `/run <文件>` - 执行 SQL 文件 ✅

## 🏗️ 技术架构

### 技术栈

```
前端：Ink 5.2.0 + React 18.3.1
语言：TypeScript 5.8.3
后端：Node.js 20+
数据库：
  - MySQL (mysql2 3.14.0)
  - PostgreSQL (pg 8.14.1)
  - SQLite (better-sqlite3 11.10.0)
测试：Jest 30.3.0 + ts-jest
```

### 模块结构

```
src/ts/
├── main.tsx              # 入口
├── app.tsx               # 主应用
├── types.ts              # 类型定义
├── cli/
│   └── commands.ts       # 命令处理
├── config/
│   └── manager.ts        # 配置管理
├── database/
```
